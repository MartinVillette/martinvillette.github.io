Forest Fire Propagation Simulation

A cellular automata-based forest fire simulation using real-world environmental data from Google Earth Engine. Developed as a TIPE project to model how wind, vegetation density, and terrain affect wildfire spread patterns.

Key Features

  • Cellular Automata Approach
    Grid-based simulation where each cell (parcel) has properties like fire state, terrain type, and environmental conditions.
  • Real-Time Visualization
    Interactive Pygame interface showing fire spread in real-time with customizable visualization modes.
  • Environmental Data Integration
    Incorporates satellite data including elevation maps, temperature readings, humidity levels, and wind patterns.
  • Fire State Modeling
    Four-stage fire progression: unburned, igniting, burning, and burned out with realistic transition mechanics.
  • Database Caching
    SQLite database system to cache Google Earth Engine queries and store simulation results for performance.
  • FIRMS Validation

Technical Implementation

The simulation uses object-oriented design with a Parcel class representing individual land units and a Map class managing the grid of parcels. Each parcel contains environmental properties like elevation, tree cover, temperature, humidity, wind speed, and direction. Fire propagation is calculated using mathematical models that consider neighboring parcels and environmental factors.
The Google Earth Engine integration provides real-world data including COPERNICUS Landcover for vegetation, USGS SRTM for elevation, NOAA GFS for atmospheric conditions, and FIRMS for fire detection. Data is cached in SQLite databases to optimize performance and reduce API calls. The simulation renders multiple visualization layers showing different environmental factors and fire progression states.

Mathematical Modeling

Fire spread rates are calculated using coefficients that account for wind effects (K_phi), terrain slope (K_theta), and vegetation density (K_s). Research Results

The simulation successfully demonstrates key fire behavior patterns: fires spread faster in the direction of wind, areas with higher tree density burn more intensely and propagate flames more rapidly, and elevation changes significantly affect spread direction and speed. Validation against FIRMS data shows good correlation between simulated and actual fire patterns.
